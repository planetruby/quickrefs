# Ruby Regular Expression Text Patterns Quick Reference (Cheat Sheet)


### Options

- /i   -  case insensitive
- /o   -  only interpolate #{} blocks once
- /m   -  multiline mode - '.' will match newline
- /x   -  extended mode - whitespace is ignored
